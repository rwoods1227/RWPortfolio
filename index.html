FantasyPay&Play

FantasyPay&Play is a fantasy football and football betting site rolled into one. It offers fully functional betting (with money in name only), where bets are paid out based on a weekly api call to fetch game results. The fantasy football side of the site allows one to create a league, add owners, draft a team, and play out games. Built with JavaScript, MongoDB, Express.js, React/Redux, Node.js, Apollo/GraphQL.

Koala

Koala is a koala-themed question and answer forum based on Quora. Users can create accounts, ask and answer questions as well as downvote and upvote questions and answers. Users can also visit thier profile page to view their questions and answers.They are also able to search for specific questions and users via the search bar. Built with JavaScript, MongoDB, Express.js, React/Redux, Node.js

Pushy Penguins

Pushy Penguins is a canvas game based off the Mario Party 5 minigame pushy penguins, as well as the story of a penguin named Grape-kun from a zoo in Japan. The goal of the game is to survive incoming waves of penguins which spawn at an increasing frequency and try to push the character off of the ice into the water. Built with vanilla JavaScript, Canvas(library), HTML, and CSS

Spotifly

Spotifly is a bug-themed Spotify clone. Many Spotify features are implemented including a dynamic search feature for instant feedback on database results. Additionally, Spotifly has a site-wide music player, so users can have the same song selected while navigating throughout the site. Built with Rails and PostgreSQL on the back-end, React and Redux on the front-end.
